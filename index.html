Anti Copy-Paste

Características Cómo Usar Instalación

Protege tu Contenido de WordPress

Plugin simple que bloquea la selección y copia de texto en tu sitio web

Características Principales

Protección de Contenido

Bloquea la selección de texto para evitar copias fáciles

Excepciones por Roles

Administradores y editores pueden copiar sin restricciones

Fácil Configuración

Se configura en minutos desde el panel de WordPress

Selectores Personalizados

Define dónde permitir selección (formularios, código, etc.)

Cómo Usar

1. Instalar y Activar

Ve a Plugins → Plugins instalados y activa "Anti Copy-Paste".

2. Configurar Opciones

Ve a Ajustes → Anti Copy-Paste y configura:

  • Activar protección global
  • Excluir roles (administradores, editores)
  • Excluir tipos de contenido
  • Definir selectores permitidos

3. Probar Funcionamiento

Abre tu sitio en modo incógnito e intenta seleccionar texto. Verás que no se puede copiar.

Instalación

Automática

  1. Ve a Plugins → Añadir nuevo
  2. Busca "Anti Copy-Paste Aprendiz de SEO"
  3. Instala y activa

Manual

  1. Descarga el archivo ZIP
  2. Sube en Plugins → Subir plugin
  3. Activa el plugin
